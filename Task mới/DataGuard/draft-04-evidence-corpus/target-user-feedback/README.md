# Target-User Feedback — Evidence Corpus

Draft 4 does not currently include a `user-study` or workbench feedback folder.
This folder therefore contains target-user feedback for the corpus
release and manuscript framing.

These files are **authored expert reviews** and must not be represented as
empirical data. They are detailed persona-based feedback artifacts intended to
help refine:

- the corpus schema;
- the release plan;
- benchmark tasks and splits;
- ethical safeguards;
- HCI/tool-builder framing.

Files:

- `EC-TOOLBUILDER-01_release_feedback.md`: feedback from a privacy NLP / HCI
  tool-builder persona.
- `EC-DATAETHICS-02_release_feedback.md`: feedback from a data ethics /
  research governance persona.
- `evidence_corpus_feedback_packet.json`: structured combined feedback packet
  covering the corpus-card fields and release expectations.
- `EC_ADDITIONAL_RELEASE_REVIEWS_03-07.md`: five additional release reviews
  from dataset-curation, legal/privacy, LLM-tooling, platform-policy, and
  replication perspectives.
